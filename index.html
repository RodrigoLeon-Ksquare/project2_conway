Instructions

  1. Select the grid size with the slider element
  2. Create a pattern. You can create manually clicking the cells in the grid or automatically clicking the random button.
  3. Start the game clicking the start game button.
  4. You can pause and continue the game.
  5. You can clear the board if you want clicking the clear button. The cells will change to "dead" status but the game will continue.
  6. If you want to reset the game, first you need to pause it and then you will can reset the game.
